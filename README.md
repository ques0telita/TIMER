# 🚀 Rocket Launch Timer
A dynamic web-based countdown timer built with HTML, CSS, and JavaScript. This project features a "Glassmorphism" interface and a dual-video system that triggers a launch sequence once the timer hits zero.

## 📌 Features
- **Custom Countdown**: Users can input a starting time for the launch sequence.

- **Step Logic**: The timer decrements by 3 units per second for a faster-paced countdown.

- **Immersive Experience**:

    -Background Video: A looping video of a rocket on the launchpad.

    -Action Video: An automatic switch to a launch/explosion video when the timer expires.

- **Auto-Reset**: Once the launch video ends, the interface automatically reverts to the initial background video.

- **Responsive Design**: Optimized for different screen sizes using CSS Media Queries.

- **Modern UI**: Glassmorphism effect with a translucent, blurred container for a high-tech feel.

## 🛠️ Technologies Used
- **HTML5*: Structure.

- **CSS3**: Flexbox, Glassmorphism (backdrop-filter), and Responsive Design.

- **JavaScript (ES6)**: Logic for setInterval, clearInterval, and event listeners for video control.
